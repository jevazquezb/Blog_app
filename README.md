# Blog app - processing data in models

> In this part of the project, we set up models for Users, Posts, Comments, and Likes. Besides, we created custom methods for each of the models.

## Built With

- Ruby on Rails & postgreSQL.
- Linters (Rubocop, Stylelint).
- Git & GitHub.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    Git
    PostgreSQL
    Ruby on Rails

### Setup

    1. git clone git@github.com:jevazquezb/Blog_app.git
    2. cd Blog_app

### Usage

    rails server
